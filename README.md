# Book Store
## Description

This project is a simple book store that was developed using a framework called Nest.
It uses Postgres as database and it's entirely built with TypeScript for better code experience.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
